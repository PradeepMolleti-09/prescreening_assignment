# Gallery Pro - Apple Styled Visual Arts Collection

A premium, high-performance visual arts curation gallery built with React and Tailwind CSS. Featuring a dark-mode theme, glassmorphism transitions, and a minimalist design language inspired by Apple’s Pro product ecosystem.

## ✨ Features
- **Apple Design Language**: Pure black palette, premium typography, and tracked-out headlines.
- **Glassmorphism Transitions**: Subtle backdrop-blur effects on navigation and interaction elements.
- **Micro-Animations**: Staggered entrance transitions and professional hover scaling.
- **Adaptive Grid**: Fully responsive layout optimizing 3-column (Desktop), 2-column (Tablet), and 1-column (Mobile) viewing.
- **Floating Details**: Curator metadata that floats cleanly over images on hover.
- **Optimized Performance**: Memoized components and custom hooks for smooth, fast rendering.

## 🛠️ Stack
- **Modern React (v19)**: Stable, performant logic.
- **Vite 6**: Next-generation frontend tooling.
- **Tailwind CSS v4**: Advanced styling and theming.
- **Custom Design System**: Bespoke CSS tokens for the "Pro" aesthetic.

## 🚀 Getting Started
1. Clone the repository
2. Install dependencies: `npm install`
3. Run locally: `npm run dev`
